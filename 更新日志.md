## [0.2.0] - 2026-02-26

### 融合升级：X/Twitter GraphQL 深度抓取

基于 [baoyu-danger-x-to-markdown](https://github.com/JimLiu/baoyu-skills/tree/main/skills/baoyu-danger-x-to-markdown) 技能融合升级，为 X/Twitter 平台新增 GraphQL 深度抓取能力。

#### 新增
- **twitter_cookies.py** — Cookie 多源管理（环境变量 → cookie 文件 → Playwright session → Chrome CDP 四源合并）
- **twitter_graphql.py** — X GraphQL API 客户端（动态 queryId 解析、TweetDetail/TweetResultByRestId 双端点、完整 feature switches）
- **twitter_thread.py** — 线程重建引擎（向上/向下分页、续页抓取、去重、根推文追溯、isSameThread 过滤）
- **twitter_markdown.py** — 线程 Markdown 渲染器（YAML front matter、编号章节、内联图片、视频链接、引用推文 blockquote）
- **四级兜底策略** — GraphQL（Tier 0）→ oEmbed（Tier 1）→ Jina（Tier 2）→ Playwright（Tier 3）
- **Schema 扩展** — `from_twitter()` 支持线程数据、内联图片、引用推文，`extra` 字段新增 `tweet_count`、`has_thread`、`images`、`videos`、`quoted_tweets`

#### 变更
- 项目从 x-reader 更名为 **feedgrab**
- `twitter.py` 调度器升级为四级兜底架构
- `.env.example` 新增 GraphQL 相关配置项（`X_AUTH_TOKEN`、`X_CT0`、`X_GRAPHQL_ENABLED`、`X_THREAD_MAX_PAGES`、`X_REQUEST_DELAY`）

#### 致谢
- 原始项目 [x-reader](https://github.com/runesleo/x-reader) by [@runes_leo](https://x.com/runes_leo)
- GraphQL 方案来源 [baoyu-danger-x-to-markdown](https://github.com/anthropics/claude-code) by [@dotey](https://x.com/dotey)
